Программа

 - загружает конфигурационный файл (формат ниже) (see standard App.config files)
 - по конфигурационному файлу находит указанные файлы (то есть в конфиге задаются полные пути к файлам, которые надо обработать)
 - заменяет в найденных файл все строки %old_value% на %new_value%
 - логирует результат (пишет в консоль) для каждого файла. file <path_from_config>: substituted <n> entries | not found | access denied | any other erro message
 - логирует общий итог: specified: <N_specified> files, found: <N_found> files, processed: <N_processed> files
 
 пример команды запуска
 
 super_program "1.3.3.3" "1.3.3.4"
 super_program - твоя программа
 "1.3.3.3" - строка %old_value%
 "1.3.3.4" - строка %new_value%
